Language Overview:

Beep Boop is a stack-based language with two keywords: "beep" and "boop". The language is case insensitive.

Running Programs:

When a Beep Boop program is run, it starts with an empty stack. Instructions can use the values on the stack as well as push or pop from the stack.

The instructions:

Stack Manipulation:

Boop

The "boop" instruction pushes the value 0 onto the stack.

Beep

The "beep" instruction pops the top value from the stack. This value is gone forever.

Boop Boop

The "boop boop" instruction pushes a value to the stack. This value is determined by reading the next line as binary. "beep" is a 1 and "boop" is a 0. This value will always be positive (no two's complement). For example, "beep boop beep" = 5.

Boop Beep

Copy the value on the top of the stack and push it onto the stack.

Output:

Beep Boop

This outputs the top of the stack as an ASCII character. Currently, Beep Boop supports all characters listed in the ASCII table on the main page (10, 32-127).

Beep Beep

This outputs the top of the stack as a base 10 number.

Control Flow:

(nothing)

No-op

Beepboop ...

Comment. Note that comments technically are instructions, so they can be jumped to. Inline comments do not exist.

Beep Beep Beep Beep

Unconditional halting of the program.

Beep Beep Boop

Unconditional jump to the line number specified by the top of the stack.

Beep Beep Beep

If the top of the stack exists and is a nonzero value, jump to the line number specified by the binary number on the next line. See "Boop Boop" for how "beep"s and "boop"s represent a binary number.

Math:

Boop Boop Boop

Pop the two numbers on the top of the stack, add them together, and push the sum to the stack.

Boop Boop Beep

Change the number on the top of the stack to its additive inverse (multiply by -1). This is one of the only ways to get negative numbers, as "boop boop" cannot read in two's complement.

Boop Beep Boop

Pop the two numbers on the top of the stack, multiply them together, and push the product to the stack.

Boop Beep Beep

Pop the two numbers on the top of the stack. Perform integer division of the first number (previously top of the stack) by the second number and push the result of the integer division to the stack first, followed by the remainder (first % second).

Registers:

Beep Boop Boop Boop

Stores the value from the top of the stack into the "boop" register.

Beep Boop Boop Beep

Stores the value from the top of the stack into the "beep" register.

Beep Boop Beep Boop

Pushes the value of the "boop" register onto the stack.

Beep Boop Beep Beep

Pushes the value of the "beep" register onto the stack.
